This repository features a compilation of Data
Engineering projects that highlight different
facets of data processing, ETL pipelines, data
warehousing, and big data technologies.
Each project is independent and illustrates
essential concepts and methodologies in the
field of data engineering. The projects
encompass data loading, preprocessing,
model training, and prediction.
Some projects have been finalized, whereas
others are currently in progress.
This Repo will be updated accordingly.
